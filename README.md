# Fitts's Law - Reaction Time Experiment

## Overview
This project is a simple web-based experiment to measure human reaction time based on **Fitts's Law**.  
The user is shown a random letter ("A" or "L") and must press the corresponding key (A for left, L for right) as quickly as possible.

This experiment was developed as part of the **Human-Computer Interaction (HCI)** lab assignment.

## Features
- Clean and responsive user interface
- 10 randomized trials
- Real-time reaction time measurement
- Displays average, fastest, and slowest reaction times
- Detailed results table showing each trial
- Option to restart the experiment
- Fully functional with keyboard input only
